Css Units

it is used to define the size of an element

they are two types of units

  1. relative units
  2. absolute units

absolute units

these are fixed units

  • px - pixel
  • cm - centimeter
  • mm - millimeter
  • in - inch
  • pt - point
  • pc - pica

relative units

  • % - percentage
  • em - em
  • rem - root em
  • vw - viewport width
  • vh - viewport height
