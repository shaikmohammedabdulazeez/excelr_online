Positions in css

to arrange elements in a positions we can use position property

  • static
  • relative
  • absolute
  • fixed
  • sticky

static

default position of an element

elements are positioned according to the normal flow of the document

top, right, bottom, left are not set

Relative

positioned relative to its normal position

top, right, bottom, left are set

z-index

z-index is used to set the stack order of an element

elements with a higher z-index are positioned in front of elements with a lower z-index

z-index only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky)

z-index is set to 0 by default and it consider positive and negative values

Fixed

positioned relative to the viewport

top, right, bottom, left are set

it is fixed to the viewport

Sticky

positioned based on the user's scroll position

it is a combination of relative and fixed

this is sticky content

absolute

positioned relative to the nearest positioned ancestor

if there is no positioned ancestor, it is positioned relative to the initial containing block

top, right, bottom, left are set

this is parent element
this is child element
