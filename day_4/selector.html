SELECTOR

universal selector

it indicates with *

it select all elements

grouping selector

it is used to select group of elements
it is indicated with comma(,)

combined selector

descendent selector

it selects all child elements (direct /indirect)

            parent child{
               // both direct and indirect childs will select
            }
child selector

it selects direct child element

                parent > child{
                 //it selects direct child
                }
adjacent sibling selector

it select immediate adjacent element

               main + adjacentelement{
                   // it selcts immediate adjacent element
               }
general sibling selector

it selects all adjacent elements

            main ~ adjacentelement{
                // it selects all siblings to the main element
            }
